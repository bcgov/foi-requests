Describe the records you are looking for:

Be as specific as possible. A good description of records will include key words, file types and the purpose that the records may have been created for.
You can include record or file numbers, or include previous Freedom of Information request numbers (if you know them), to help us find the right records. Please do not use any special characters, including quotation marks or bullet points.
You can browse the Open Information catalogue to see examples of other requests. For example: Briefing notes prepared for the minister on a specific topic. Requesting - Any and all records - can result in a high volume of unnecessary records.

What time frame should we search?

Be as specific as possible. Limiting the time frame of your request will help us get the records to you faster.
